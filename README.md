# My Game of Life

My Game of Life is a Java implementation for [Conway's Game of Life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

## Run Tests

Use the Gradle Wrapper to run the tests.

```bash
./gradlew test
```

## Usage

Use the Gradle Wrapper to run the tests.

```bash
./gradlew run
```
